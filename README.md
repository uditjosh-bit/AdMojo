# 🚀 AdMojo AI - Ad Copy Generator

## ✨ Overview
The easiest way to use this repository is to let an AI Assistant (like Claude, ChatGPT, or Gemini) run the process for you. You don't need to install Python or run any complicated terminal commands yourself!

## 🤖 How to Run (AI-Assisted)
1. Open your favorite AI Assistant.
2. Give it the link to this GitHub repository (or copy/paste the `App.jsx` file).
3. Copy and paste the exact prompt below.

## 📋 The Magic Prompt
> "I want to run the React application found in this repository's `App.jsx` file. Please give me the exact step-by-step terminal commands to set up a new React project using Vite, install the required dependencies (including `lucide-react` and `tailwindcss`), replace the default App component with this code, and start the local development server."
